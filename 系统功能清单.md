# 非遗传承信息管理系统 - 功能清单

## 📱 用户端功能

### 1. 用户认证模块
- **用户注册**
  - 填写用户名、密码、昵称、邮箱、手机号
  - 密码强度验证
  - 邮箱/手机号格式验证
  - 用户名唯一性检查
  
- **用户登录**
  - 用户名/邮箱/手机号登录
  - 密码加密验证
  - JWT Token生成
  - 记住登录状态
  - 自动登录功能
  
- **密码管理**
  - 修改密码
  - 密码强度提示

### 2. 首页模块
- **轮播图展示**
  - 多张轮播图自动切换
  - 点击跳转到详情页
  - 响应式适配
  
- **统计数据展示**
  - 非遗项目总数
  - 传承人总数
  - 活动总数
  - 媒体资源总量
  
- **精选内容展示**
  - 精选非遗项目（4个）
  - 热门活动（4个）
  - 最新资讯（5条）
  - 通知公告（6条）
  
- **快速入口**
  - 非遗项目入口
  - 传承人入口
  - 活动预约入口
  - 文创商城入口

### 3. 非遗项目模块
- **项目列表**
  - 分页展示所有项目
  - 按分类筛选
  - 按级别筛选（国家级/省级/市级）

- **项目详情**

  - 传承现状描述
  - 保护单位信息
  - 申报日期
  - 相关传承人列表
  
- **项目地图**
  - 地理位置可视化展示
  - 按地区查看项目分布
  - 按分类查看项目分布
  - 地区统计数据
  - 项目标记点击查看详情
  
- **项目搜索**
  - 全文搜索
  - 搜索建议（自动补全）
  - 热门搜索推荐

### 4. 传承人模块

### 5. 活动预约模块

- **活动详情**
